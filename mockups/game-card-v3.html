Game card — tweaks + the scaled-row idea

Set A = small tweaks to what we have (fixing the heavy full-bleed band). Set B = distribution built into each team's row, scaled so 100% = the week's biggest single-team pick count (here 173, Texas A&M). Bar length shows pick volume vs. the whole slate; the gold part of each bar = that team's locks.

Set B · distribution inside each row, scaled to the week's max (173)

Each variant shows a high-volume game (Texas, 279 picks) and a low-volume game (Virginia, 87 picks) using the same scale, so you can see the bars get shorter when fewer people picked the game. Gold = locks, brown = non-lock picks; full track = 173.

A thin scaled gauge sits under each team's name. Length = picks vs. the week max; gold leading segment = locks. Count · locks at the row's right.
